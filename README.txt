HOW TO USE Ge'ez Key Mapper (GKM)

1. GKM is a System Tray application

2. After installation, run GKM.exe

3. A tray icon will appear which can
   be left-clicked to toggle between
   "Amharic (Ge'ez)" and "English" modes

4. A right-click on the tray icon will
   bring up a menu with "Key Mappings ...",
   "About ..." and "Exit" options
